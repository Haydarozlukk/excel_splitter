# Excel Splitter (Python)

This script splits large Excel `.xlsx` files into smaller chunks (default: 400,000 rows per file).

## Usage

```bash
python bol_excel.py
```

Then enter the path to your folder containing Excel files.

## Requirements

- Python 3.10+
- pandas
- openpyxl

## Installation

```bash
pip install -r requirements.txt
```
